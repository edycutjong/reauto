## Project Description *
```
Reauto is an AI-powered cross-chain exploit forensics tool that traces stolen funds across multiple blockchains and generates human-readable "death certificates" for DeFi hacks.

The problem: When a DeFi exploit happens, analysts manually trace transactions across block explorers. The Wormhole hack moved $320M across 4 chains in 23 minutes. No unified tool shows the full attack timeline.

How it works:
1. Input — Paste any hack transaction hash
2. GoldRush Trace — Covalent's GoldRush API traces the money flow across chains, resolving token transfers, contract interactions, and bridge hops
3. AI Narration — OpenAI synthesizes the transaction graph into a human-readable timeline: "At 14:23 UTC, attacker exploited reentrancy in Contract 0xABC, draining 12,000 ETH..."
4. Death Certificate — A forensic report card showing: stolen amount, attack vector, affected chains, laundering path, and time-to-drain

GoldRush is essential because it provides unified cross-chain transaction data through a single API. Without GoldRush, you'd need separate indexers for each chain plus custom normalization logic.

Tech stack: Next.js 16 (App Router), React 19, Tailwind v4, GoldRush API (Covalent), OpenAI API. 100% test coverage. Built solo.
```
